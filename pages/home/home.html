Meu objetivo com este projeto é treinar e aplicar todas as funcionalidades do chamado C.R.U.D. (Create, Read, Update, Delete), um acrônimo para as operações realizadas em informações armazenadas.
Create (Criar): Refere-se à operação de inserir novos dados no sistema. Read (Ler): Consiste na operação de recuperar dados existentes do sistema. Update (Atualizar): Esta operação modifica dados existentes. Delete (Deletar): Refere-se à remoção de dados do sistema.
